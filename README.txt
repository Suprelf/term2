Програма генерує природний текст статистичними методами. Для виконання свого завдання, програма повинна отримати аргументи:
- розмір початкової урни (в словах, оптимальне значення - 5)
- обсяг тексту (в словах, довільний)
- Ro (параметр, що відповідає кількості доданих "старих" слів на кожній виборці з урни)
- Nu (параметр, що відповідає кількості доданих "нових" слів на кожній виборці з урни)
Після завершення генерації тексту з заданими параметрами, програма дозволяє переглянути його як масив, словник тексту, вивести його в файл Created_Text.txt.
